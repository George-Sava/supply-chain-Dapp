---Description---
    Simple Supply-Chain Dapp for Ethereum blockchain. 
    
    Devstack: React, Redux, Drizzle, Web3, Solidity, MUI, Webpack5  

---Necesary steps for starting the Development local blockchain server---    
    1. Download and install Ganache --> https://trufflesuite.com/ganache/index.html
    2. Download and install Truffle --> https://trufflesuite.com/truffle/
    3. Git clone repository
    4. Open terminal in project root folder
    5. Migrate smart contracts with command: truffle migrate . You can compile the smart contracts before migrating the to see if there are any compile errors.
    6. Run: cd client
    7. Finally: npm start , to start React
